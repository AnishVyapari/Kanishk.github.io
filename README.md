## Configuration

To customize the portfolio, edit the `CONFIG` object in `kanishk.html`:

```javascript
const CONFIG = {
  githubUser: "your-github-username",
  name: "Your Name",
  role: "Your Role",
  bio: "Your bio",
  email: "mailto:your-email@example.com",
  image: "./your-image.jpg",
  education: {
    college: "Your College",
    degree: "Your Degree",
    startYear: 2025
  }
};
```

## Deployment

This portfolio is deployed using **GitHub Pages** with automatic workflows:

1. Push changes to the `main` branch
2. GitHub Actions workflow triggers automatically
3. Static files are deployed to GitHub Pages
4. Site is live at `https://[username].github.io/Kanishk.github.io`
